# рҹ“– About the Project

**Gantabya** is a modern tourism platform designed to simplify travel planning across Nepal. Instead of searching multiple websites for destinations, accommodations, transportation, and travel information, users can discover everything in one intuitive platform.

The project focuses on delivering a seamless travel experience through an attractive user interface, personalized destination recommendations, and easy trip planning.

Whether users are planning a weekend getaway or a long vacation, Gantabya provides the information needed to make informed travel decisions.
